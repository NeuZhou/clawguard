# 仪表盘使用指南

## 访问

默认地址：**http://localhost:19790**

## 标签页

| 标签 | 功能 |
|------|------|
| 概览 | 健康状态、活跃会话、今日费用、安全告警 |
| 监控 | 实时消息流（SSE）、会话筛选 |
| 安全 | 安全评分（0-100）、OWASP 覆盖、规则管理 |
| 费用 | 日/周/月费用分析、预算状态、预测 |
| 审计 | 哈希链事件日志、完整性验证、导出 |
| 设置 | 预算、告警、导出器、规则开关 |

## REST API

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/overview` | GET | 概览统计 |
| `/api/messages` | GET | 消息记录 |
| `/api/security/score` | GET | 安全评分 |
| `/api/cost` | GET | 费用明细 |
| `/api/audit/verify` | GET | 哈希链验证 |
| `/api/stream` | GET | SSE 实时流 |
