# 常见问题

**Q: 会影响 Agent 性能吗？**
不会。安全扫描在微秒级完成，存储是追加写入的 JSONL。

**Q: 需要数据库吗？**
不需要。纯文件存储，自动轮转和 gzip 压缩。

**Q: 支持 Windows 吗？**
支持。零原生依赖，Node.js 能运行的地方就能用。

**Q: 检测到密钥泄露会怎样？**
生成告警，包含密钥类型、脱敏证据和提供商的密钥轮换 URL。

**Q: 能导出到 Splunk/ELK 吗？**
可以。启用 syslog 导出器（RFC 5424）或 webhook 导出器。

**Q: Token 估算准确吗？**
使用 `ceil(text.length / 4)` 估算，对英文文本大致准确。中文文本可能偏差较大。

**Q: GitHub Copilot 模型怎么计费？**
按 $0 追踪（订阅包含）。
